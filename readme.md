# algolithm

Simple repository of algorithm implementations, utilities, and examples.

## Contents
- src/ — algorithm implementations
- examples/ — usage examples
- tests/ — unit tests

## Getting started
1. Clone: `git clone <repo-url>`
2. Open the project folder: `cd algolithm`

## Usage
- Run examples or specific implementations in the language/runtime used (see each subfolder for instructions).
- Run tests: `./run-tests` or check test README in `tests/`.

## Contribution
- Fork, create a branch, add tests for new code, open a PR.
- Follow existing style and include brief description in PR.
